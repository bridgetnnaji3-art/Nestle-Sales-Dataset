# Nestle-Excel-Dataset with Dashboard
This repository contains a cleaned dataset of Nestle products along with an Excel dashboard for visualization and analysis.
Nestle is a top global food and beverage company focused on quality and innovation. I work as a data analyst, cleaning and analyzing product data, uncovering insights, and building dashboards to help guide business decisions

## Contents
- 'data/raw/' - Original dataset files.
- 'data/cleaned/' - Cleaned and formatted Excel datasets.
- 'dashboard/' -  Excel dashboard with charts, pivot tables, and slicers.

  ## Features
- Cleaned dataset with missing values handled and standardized columns.
- Interactive Excel dashboard showing:
    - Product sales  trends
    - Category performance
    - Geographic insights
- Ready-to-use for analysis or reporting.

  ## How to Use
  1. Open 'Nestle Sales Dashboard Assignment.xlsx' to explore charts and filters
  2. Check 'Nestle Sales Data Assignment.xlsx' for the processed dataset.
